# 🏠 College Hostel Management System

A web-based platform designed to **digitize and streamline hostel management operations**, replacing manual processes with an efficient and centralized system.

---

## 🚀 Features

### 👨‍💼 Admin Panel

* Add / Delete student records
* Manage hostel database
* Secure authentication login system
* Post announcements
* View and manage leave applications
* Handle and resolve complaints

### 👩‍🎓 Student Panel

* View announcements
* Access room allocation details
* Submit complaints (electricity, plumbing, carpenter, etc.)
* Track complaint status
* Apply for leave online

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL

---

## 🎯 Objective

To eliminate manual paperwork in hostel management by providing a **centralized digital solution** that improves efficiency, transparency, and communication between students and administration.

## ⚙️ Installation & Setup

1. Clone the repository

   ```bash
    git clone https://github.com/Shiiivaniii/Hostel-Management-Website.git
   ```

2. Move the project to your XAMPP `htdocs` folder

3. Start Apache and MySQL from XAMPP

4. Import the database:

   * Open phpMyAdmin
   * Create a database
   * Import the `.sql` file

5. Run the project in browser:

   ```bash
   http://localhost/sl_project
   ```
